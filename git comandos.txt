git 1. Verificar o Status
   Comando: git status

2. Adicionar Arquivos ao Stage
   Comando:
   - git add <arquivo> (Adiciona um arquivo específico)
   - git add . (Adiciona todos os arquivos modificados)

3. Fazer um Commit
   Comando: git commit -m "Descrição das alterações"

4. Enviar o Commit para o Repositório Remoto
   Comando: git push origin main/master

5. Verificar se Tudo Foi Enviado
   Comando: git log --online

6. Trabalhando com Múltiplos Branches (Opcional)
   Comando: git push origin <nome_do_seu_branch>

7. Atualizando o Código Local
   Comando: git pull origin main
